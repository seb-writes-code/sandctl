# sandctl (TypeScript + Bun)

This directory contains the TypeScript rewrite of `sandctl`.

## Prerequisites

- Bun 1.x

## Build

```bash
cd sandctl-ts
bun install
bun run build
```

## Install

```bash
make install
```

## Cross-compile

```bash
make build-all
```

## Quick Start

```bash
cd sandctl-ts
bun install
bun run build
./sandctl --help
./sandctl version
```
